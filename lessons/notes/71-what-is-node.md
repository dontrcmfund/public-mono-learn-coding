# What is Node.js? (first principles)

Goal: understand what Node.js is and why it exists.

Why do we care?
- It lets you run JavaScript outside the browser
- You can build tools, servers, and scripts with JS

First principles
- Browsers run JavaScript on web pages
- Node.js is a program that runs JavaScript on your computer
- It gives you access to files, folders, and the operating system

Why this matters to you (real life)
- You can build small tools for yourself
- You can automate tasks with JavaScript
- Many modern apps use Node for servers

If all you remember is one thing
- Node.js lets JavaScript run outside the browser
